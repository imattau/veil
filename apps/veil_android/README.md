# veil_android

VEIL Android client built with Flutter and the local `veil_sdk` package.

## Getting Started

This project uses the workspace SDK at `packages/veil-sdk-dart`.

### Run (Android)

```bash
cd apps/veil_android
flutter pub get
flutter run
```
